## 🌌 Overview

**Veil of Fate** is an original **story-based cinematic game** built in **Unity (C#)**, blending deep narrative storytelling, stylized visuals, and intense character-driven gameplay.

The game follows a secret agent entangled in a global conspiracy — uncovering the truth behind a mysterious organization bent on reshaping the world order.

This repository serves as the **official portfolio site** for the project, built with **React + Vite + Tailwind CSS**.

> **Current Version:** `v1.0.0` — Character design & teaser visual phase

---

## 🧠 Concept & Storyline

Set in a dystopian world filled with corruption, betrayal, and hidden agendas, **Veil of Fate** explores the fine line between heroism and sacrifice.

The narrative follows the **Protagonist**, an undercover agent who uncovers a vast syndicate after a devastating event. Alongside a key supporting character, he battles not just the syndicate — but his own doubts and moral conflicts — leading to an unforgettable final showdown.

> *"In this world, fate isn't written — it's veiled."*

## 🛠️ Running the Portfolio Site Locally

```bash
# Clone the repository
git clone https://github.com/Kiran7613/game-portfolio.git
cd game-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
